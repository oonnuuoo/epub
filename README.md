# ZIP to EPUB Converter

画像入りZIPファイルをブラウザ上でEPUBファイルに変換するWebアプリケーションです。

## 機能

- ZIPファイル内の画像を自動的に抽出してEPUBに変換
- ドラッグ&ドロップ対応
- カバー画像の自動生成
- 右綴じ（RTL）対応
- 完全にブラウザ上で動作（サーバー不要）

## 使い方

1. ZIPファイルをアップロード（クリックまたはドラッグ&ドロップ）
2. 書籍のタイトル、著者名、出版社を入力
3. 「EPUBに変換」ボタンをクリック
4. 生成されたEPUBファイルをダウンロード

## GitHub Pagesでのデプロイ

1. このリポジトリをGitHubにプッシュ
2. リポジトリの Settings > Pages
3. Source で `main` ブランチを選択
4. 公開されたURLにアクセス

## 技術スタック

- HTML5 / CSS3
- JavaScript (ES6+)
- JSZip - ZIPファイルの処理とEPUB生成

## 注意事項

- すべての処理はブラウザ内で完結します（ファイルはアップロードされません）
- 大きなZIPファイルの場合、処理に時間がかかる場合があります
- 生成されるEPUBは右綴じ（日本語漫画向け）の設定になっています
